A full-stack web application that enables users to report incidents and visualize them on an interactive map in real-time. Built with a scalable architecture using modern technologies.
Features
🔐 User Authentication
Secure Login & Registration using JWT
🗺️ Interactive Map
View reported incidents with location-based markers
📍 Incident Reporting
Users can report crimes/events with details
⚡ Real-Time Updates
Dynamic data rendering on map
🧩 Modular Backend Architecture
Organized into routes, services, models, utils
🛡️ Secure API Design
Token-based access control
📦 Scalable Database
MongoDB with async operations (Motor)
