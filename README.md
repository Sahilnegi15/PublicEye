A full-stack real-time incident reporting and monitoring platform built using FastAPI, React.js, MongoDB, and MySQL. PublicEye enables users to report incidents, track events geographically, and analyze structured and unstructured data through scalable APIs and interactive dashboards.

---

## 🚀 FEATURES

- 📍 Real-Time Incident Reporting
  - Report incidents with geolocation support
  - Live data ingestion and updates
  - Real-time event monitoring

- 🌍 Geospatial Querying
  - Location-based incident search
  - Efficient geospatial indexing using MongoDB
  - Nearby incident detection

- ⚡ Scalable Backend Architecture
  - Modular FastAPI backend
  - Async database operations using Motor
  - High-performance REST APIs

- 🗂️ Structured & Unstructured Data Support
  - MongoDB for flexible document storage
  - MySQL for relational data management

- 📊 Interactive Dashboard
  - React-based frontend
  - Incident visualization and analytics
  - Real-time updates and filtering
